Kitabalaya: A Library Management App

TKitabalaya is a lightweight, browser-based library management application designed to digitize and simplify book inventory operations for small to mid-sized libraries. Built with performance and ease of use in mind, the system replaces traditional manual ledger tracking with a clean, intuitive interface that requires no backend server, setup, or external dependencies. The application follows a streamlined CRUD architecture, enabling users to efficiently add, view, update, and remove books with persistent client-side storage. A minimal yet informative dashboard provides real-time insights into inventory size, availability, and borrowing status, ensuring quick decision-making without unnecessary complexity

Impact: By combining instant search, genre-based filtering, and status-based organization, Kitabalaya minimizes overhead and significantly reduces the time required to manage records—all while maintaining a fast, responsive, and user-friendly experience.

Technical Highlights:

  • Lightweight, dependency-free architecture using vanilla JavaScript, HTML5, and CSS
  • LocalStorage-based persistence, eliminating backend and database overhead
  • Full CRUD operations with real-time UI updates
  • fast client side search and filterin by title, author, genre and availability.
  • Responsive, mobile-friendly design optimized for low-resource environments

Tech Stack

Vanilla JS HTML5/CSS3 LocalStorage

The WCA World Championship 2025 was more than just a competition—it was an emotional journey, a celebration of global talent, and a personal adventure.
